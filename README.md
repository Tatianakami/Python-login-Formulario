🚀 Como Executar (Atualizado)
bash
# Clone o repositório
git clone https://github.com/seu-usuario/sistema-login-pdf.git
cd sistema-login-pdf

# Instale as dependências (incluindo novas)
pip install PyMuPDF pillow pandas customtkinter

# Execute o sistema
python app.py
